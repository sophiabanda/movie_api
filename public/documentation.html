Sophia's Movie Database

API Documentation

BUSINESS LOGIC URL HTTP METHOD REQUEST BODY RESPONSE BODY DATA FORMAT
Get a list of all films /films GET None A JSON object containing the list of films and their data
Get films by title /films/[Film Title] GET None A JSON object containing a specified film and its data.
Get films by genre /films/genre/[Name of Genre] GET None An array of JSON objects containing films with the specified associated genre.
Get films by director name /films/director/:directorName GET None A JSON object returning a list of films by the specified director.
Get a list of all registered users /users GET None An array of JSON objects contianing user data
